Class: SVGWorld

SVGWorld()

SVGWorld - Objeto "mundo" centralizado para manejo de SVG Resuelve el problema de transformaciones entre múltiples sistemas de coordenadas: 1. Coordenadas SVG (viewBox) - El espacio de coordenadas del SVG original 2. Coordenadas del viewport (pan/zoom) - La transformación aplicada por el usuario 3. Coordenadas de pantalla - Las coordenadas del navegador donde ocurren eventos Esta clase actúa como un objeto intermedio "en memoria" que mantiene la representación del estado del SVG y proporciona métodos para conversión de coordenadas y manipulación de elementos de forma unificada.
