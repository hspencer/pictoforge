Class: PathDataProcessor

PathDataProcessor(pathString)

PathDataProcessor Service Servicio centralizado para manipulación de paths SVG usando svg-pathdata. Proporciona acceso directo a puntos de control Bézier (C1, C2, Q1) y puntos de anclaje, permitiendo edición precisa de curvas. Características: - Parser de cadena 'd' a AST (Abstract Syntax Tree) - Normalización a comandos absolutos - Acceso directo a puntos de control y anclaje - Modificación de segmentos Bézier - Regeneración de comandos Path - Utilidades geométricas (inversión, cálculos)
